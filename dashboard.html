Tải File Landing Page

Hãy tải file index.html lên để bắt đầu chỉnh sửa nội dung.

Chưa nhận file

Kéo & thả file index.html vào đây

Hoặc nhấn vào đây để duyệt file từ máy tính của bạn.

Hướng dẫn sử dụng

1. Chọn file `index.html` trong thư mục dự án của bạn.
2. Chỉnh sửa văn bản, đường dẫn và hình ảnh tại các tab tương ứng.
3. Nhấn "Xuất File mới" ở cuối trang để tải xuống file đã chỉnh sửa.
